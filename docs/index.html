VT100 emulation resources collection

This page lists some files and sites I found helpful when developing ESPTerm.

Web references

  • An invaluable resource in my research was the website vt100.net which provides original manuals for many VT models.
  • The Xterm manual page also proved immensely useful. I even made a copy of it with highlighted differences and missing features in ESPTerm.
  • Some sequences can also be found in the GNU Screen manual page
  • Application Keys Mode became clear after reading this page: www.tldp.org/HOWTO/Keyboard-and-Console-HOWTO-21.html.
  • When trying to understand what codes should be sent by function and control keys, this page provided some great insight: invisible-island.net/xterm/xterm-function-keys.html
